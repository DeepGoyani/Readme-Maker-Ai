## 🚀 Quick Start

### 📦 **Installation**

<div align="center">

```bash
# 🔄 Clone the repository
git clone https://github.com/username/Readme-Maker-Ai.git
cd Readme-Maker-Ai

# 📦 Install dependencies
npm install

# 🔧 Copy environment variables
cp .env.example .env

#  Start development server
npm run dev
```

</div>
