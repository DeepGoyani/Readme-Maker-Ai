# README Maker AI

An AI-powered README generator that helps you create beautiful, professional documentation for your projects.

## Features

- AI-powered content generation
- Multiple templates and themes
- Real-time preview
- Export to Markdown
- Easy customization

## Tech Stack

- React + TypeScript
- Vite
- Tailwind CSS
- shadcn/ui components
- Framer Motion

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

## Project Structure

- `src/` - Source code
- `public/` - Static assets
- `components/ui/` - UI components
